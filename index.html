Tentang Saya

Saya adalah mahasiswa dengan konsentrasi Rekayasa Perangkat Lunak yang memiliki ketertarikan besar pada dunia Software Engineering, Artificial Intelligence, dan Data Science. Saya menguasai berbagai bahasa dan teknologi seperti HTML, CSS, C++, Python, SQL, dan Java OOP, serta berpengalaman menggunakan framework React dan Laravel dalam pembuatan website. Selama kuliah, saya telah mengerjakan beberapa proyek seperti website perpustakaan dan website Kupi Kuki, serta terbiasa menggunakan VSCode, Git, Figma, dan XAMPP. Saya adalah pribadi yang detail-oriented, perfeksionis, dan bertanggung jawab, sering mengambil peran dalam mengatur tugas tim dan merapikan hasil akhir proyek agar tampilan dan fungsionalitasnya optimal

Saya menyukai interaksi antara manusia dan komputer, terutama dalam hal desain visual dan pengalaman pengguna (UI/UX), namun juga berkomitmen untuk memperdalam kemampuan backend. Saya memiliki kemampuan komunikasi, kepemimpinan, dan manajemen anggota yang saya kembangkan melalui pengalaman organisasi, serta terus berlatih dalam public speaking dan leadership. Di luar bidang akademik, saya menikmati bertemu orang baru dan bermain alat musik, dengan cita-cita untuk menjadi profesional di bidang teknologi yang dapat terus belajar dan berkembang.
